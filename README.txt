Pasirinkta programavimo kalba - PHP
Reikia leisti scriptą per serverį, galima per localhost. Kitaip neveiks.
Atliktos visos užduoties dalys, programa sugeneruoja du .csv failus: viename yra pastovi palūkanų normą, o kitame pasikeičianti.
Duomenis galima perduoti per HTML formą.

Detalesnė paleidimo instrukcija, siūlomas paleidimo būdas:
1. Įsirašyti XAMPP įrankį į kompiuterį;
2. Į XAMPP kataloge esantį "htdocs" katalogą įmesti "loan_calculator";
3. Naršyklės pagalba užeiti į http://localhost:8080/loan_calculator/ ;
4. Įvesti pageidaujamus duomenys;
5. Rezultatas sugeneruojamas šakniniame kataloge "loan_calculator"
